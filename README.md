# Bài tập Unit Test - Student Analyzer

## Mô tả
Chương trình hỗ trợ phân tích điểm số học sinh với các chức năng:
1. Đếm số lượng học sinh giỏi (Điểm >= 8.0).
2. Tính điểm trung bình của lớp (Chỉ tính các điểm hợp lệ từ 0-10).

## Yêu cầu kỹ thuật
- Java JDK 11+
- Thư viện JUnit 5 (Jupiter)

## Cấu trúc thư mục
- `src/`: Chứa mã nguồn chương trình (`StudentAnalyzer.java`)
- `test/`: Chứa mã kiểm thử (`StudentAnalyzerTest.java`)

## Hướng dẫn chạy Test
Để chạy kiểm thử, bạn cần thêm thư viện JUnit vào classpath hoặc chạy trên IDE (IntelliJ/Eclipse/VS Code):

1. Mở Project.
2. Chuột phải vào file `StudentAnalyzerTest.java`.
3. Chọn **Run 'StudentAnalyzerTest'**.
4. Kiểm tra kết quả xanh (Passed) ở thanh trạng thái.
